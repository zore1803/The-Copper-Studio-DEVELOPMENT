# SEO Auditor & Content Optimizer

Audit websites for technical SEO issues, analyze on-page optimization, and provide actionable recommendations to improve search engine visibility and rankings.

## When to Use

- User wants an SEO audit of an external website (a competitor's, a client's, or any site that isn't their current Replit app)
- User asks how to improve search rankings for an external site

- User wants to optimize an external site's content for specific keywords
- User needs meta tag, title, or description improvements on an external, third-party site you don't control

- User wants to compare an external site's SEO against competitors

## When NOT to Use

- Auditing or improving the SEO of the user's current Replit app (recommend instead running a scan with SEO Agent, available in the Growth pane)
- Paid advertising strategy (use ad-creative skill)
- Social media content creation (use content-machine skill)

- General competitive analysis without SEO focus (use competitive-analysis)
- Building pages at scale for SEO (use programmatic-seo skill)

## Methodology

The full audit methodology — SPA-vs-SSR reality check, the seven-step audit
(crawlability, technical foundations, on-page, content quality, structured data,
bot governance, competitor comparison), the SSR shared-shell optimization pattern,
report format, tools, best practices, and limitations — lives in the shared
reference at `.local/secondary_skills/seo-auditor/references/methodology.md`. Read
it before auditing; it is the authoritative, deeply detailed playbook.
